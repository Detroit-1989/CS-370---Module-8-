# CS-370---Module-8-

For this project, I developed an intelligent pirate agent using reinforcement learning. I was provided with starter code that included the game environment, the maze layout, helper functions, and portions of the deep Q-learning framework. The starter code established the structure of the project and allowed me to focus on implementing the learning algorithm.

The majority of my work involved completing the Deep Q-Learning implementation. I created the code that trains the pirate agent by allowing it to explore the maze, store experiences in memory, and learn from those experiences using a neural network. I also configured the training loop, adjusted important hyperparameters such as the exploration rate and discount factor, and tested the model until the pirate consistently found the treasure. This project demonstrated how reinforcement learning can solve complex decision-making problems without explicitly programming every possible action.
